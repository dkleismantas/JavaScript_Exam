# How to run the assignments

**Warning:** All tasks in the Word file are provided in Lithuanian.

- Clone the repository. Inside, you will find two files with the extensions **.js** and **.txt**, which contain the solutions, and a third **Word** file with the tasks.
- Open the **.txt** file with **Notepad** or the **.js** file with **VS Code**.
- Copy the solution of the task and paste it **[here](https://jsbin.com/?js,console)** or **[here](https://playcode.io/empty-javascript)** *(In the first link, you must press **Run** to execute the code; in the second link, the code runs automatically.)* (You do not need to copy the task **numbering**).
- The result will be visible in the console window on the right.
